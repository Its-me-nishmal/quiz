# Installation Guide

## Prerequisites
- Node.js
- MongoDB
- Git

## Steps
1. Clone the repository:
    ```sh
    git clone <repo-url>
    ```
2. Install dependencies:
    ```sh
    cd project-root
    npm install
    ```
3. Set up environment variables:
    ```sh
    cp .env.example .env
    ```
4. Start the development server:
    ```sh
    npm run dev
    ```
